What is hmOS?

Based on the operating system theory, it provides a universal human-machine computing architecture. Make full use of the computing and storage capabilities of machines, as well as the unique reasoning and judgment capabilities of human beings.Human and machine complete a complex task together. It mainly consists of task management, resource management, human-machine interface and system monitoring.

Flexibility

Support user tasks' secondary application development.

Extensibility

Abstract internal implementation details to improve scalability.

Universality

Allowing a wider range of users to use the platform, the human + intelligent machine model can liberate them from the tedious process.

Major Functions

Our operating system provides.
Provide human and machine computing resources, virtualize computing resources.
Provide the framework and algorithm of task allocation.
Provide human-machine cooperation mode.
